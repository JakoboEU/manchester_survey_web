# City Nature Choices – Frontend + Supabase backend

This repository contains:

- A **React + Vite** frontend, built into `/docs` and hosted via GitHub Pages.
- A **Supabase backend** (Postgres, Auth, Edge Functions) managed via the Supabase CLI in `/supabase`.

---

## 1. Prerequisites

You’ll need:

- **Node.js** ≥ 20 (for Vite & Supabase CLI)
- **npm**
- **Docker Desktop** (or compatible engine like Colima/Podman/Rancher) for local Supabase
- A **Supabase account** and at least one project (for production)
- Install **supabase cli** as per https://supabase.com/docs/guides/local-development
